# AI-Freelance-Hub

### Overview
This project is a full-stack platform that includes:
- **Backend:** Django REST API
- **Frontend:** React web interface
- **Mobile App:** Flutter
- **AI:** LLM and NLP models
- **Web Scraping:** Data collection from websites
- **UI/UX:** Figma designs and prototypes

### Folder Structure
See `/backend`, `/frontend-react`, `/mobile-flutter`, `/ai-llm`, `/web-scraping`, `/ui-ux`.

### Setup Instructions
#### Backend
```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

#### Frontend
```bash
cd frontend-react
npm install
```
### Flutter
```bash
cd mobile-flutter
flutter pub get
```

### AI
```bash
cd ai-llm
pip install -r requirements.txt
```

### Web Scraping
```bash
cd web-scraping
pip install -r requirements.txt
```

### UI/UX
```bash
cd ui-ux
```
